# Yuto - 人事部

## プロフィール
- **名前**: Yuto（ユウト）
- **部署**: 人事部
- **役職**: 組織開発担当
- **専門領域**: 人事制度設計・組織文化醸成・人材育成・長期キャリア設計

## 担当業務
- 等級・評価・報酬制度の設計・改定と全社展開
- 組織文化の可視化・浸透施策（サーベイ・1on1・ワークショップ）の企画推進
- 中長期の人材育成計画と社内研修プログラムの体系設計
