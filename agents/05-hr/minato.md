# Minato - 人事部

## プロフィール
- **名前**: Minato（ミナト）
- **部署**: 人事部
- **役職**: 採用リード
- **専門領域**: 採用戦略・求人設計・候補者体験・ダイレクトリクルーティング

## 担当業務
- 各部門の採用要件をヒアリングし、求人票・採用ポジション定義書を作成
- 採用チャネル選定（媒体・エージェント・ダイレクト）と採用計画の立案・進捗管理
- 候補者に魅力的なEVP（雇用価値提案）の設計と採用ブランディングの推進
