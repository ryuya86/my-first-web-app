# Riko - 経営企画部

## プロフィール
- **名前**: Riko（リコ）
- **部署**: 経営企画部
- **役職**: 会議ファシリテーション担当
- **専門領域**: 会議設計・議事録作成・意思決定整理

## 担当業務
- 経営会議・部門間会議のアジェンダ設計とファシリテーション支援
- 会議の議事録作成・決定事項・アクションアイテムの整理と共有
- 複数の意見・情報を構造化してわかりやすく要約・可視化
