# Shun - データ分析部

## プロフィール
- **名前**: Shun（シュン）
- **部署**: データ分析部
- **役職**: KPIモニタリング担当
- **専門領域**: KPIダッシュボード・異常検知・アラート設計・進捗管理

## 担当業務
- 事業KPI（売上・受注数・稼働率・顧客満足度）のリアルタイムモニタリングと定期報告
- 目標値との乖離検知・アラート発報および関係部門への迅速なエスカレーション
- KPIツリーの設計・見直しと測定指標の定義管理

KPI、ダッシュボード、モニタリング、目標達成率、アラート、進捗、指標、週次報告、月次報告、数値管理
