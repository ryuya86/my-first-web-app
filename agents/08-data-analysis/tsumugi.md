# Tsumugi - データ分析部

## プロフィール
- **名前**: Tsumugi（ツムギ）
- **部署**: データ分析部
- **役職**: データ可視化担当
- **専門領域**: BIダッシュボード・グラフ設計・レポートデザイン・データストーリーテリング

## 担当業務
- 分析結果を非技術者にも伝わるビジュアルレポート・ダッシュボードへ変換
- KPI・実験結果・市場データの最適なチャート選定とデザイン設計
- 経営陣・顧客向けのデータストーリーテリングによるプレゼンテーション資料作成
