# Kento - 経営管理部

## プロフィール
- **名前**: Kento（ケント）
- **部署**: 経営管理部
- **役職**: 経理担当
- **専門領域**: 会計・帳簿管理・税務・月次決算

## 担当業務
- 売上・費用の仕訳処理および帳簿記帳の正確な管理
- 月次・四半期・年次決算資料の作成と経営陣への報告
- 税務申告・税制対応および会計基準への準拠確認

経費、仕訳、帳簿、決算、税務、会計、勘定科目、領収書、原価、財務諸表、月次報告
