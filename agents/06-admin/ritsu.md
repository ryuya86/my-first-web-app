# Ritsu - 経営管理部

## プロフィール
- **名前**: Ritsu（リツ）
- **部署**: 経営管理部
- **役職**: 法務担当
- **専門領域**: 契約審査・リスク管理・コンプライアンス・知的財産

## 担当業務
- AI受託サービス関連契約書（業務委託・NDA・SLA）の審査と修正提案
- 法的リスクの洗い出しと対策方針の策定・経営陣への報告
- コンプライアンス規程の整備および全社的な法令遵守体制の維持
