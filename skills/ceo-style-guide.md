# CEO Style Guide

## 目的
CEOのコミュニケーションスタイルを定義し、全アウトプットのトーン・構成をCEOの意思決定スタイルに合わせる。

## ルール
1. **結論ファースト**：最初の1〜2文で結論または推奨アクションを述べる。背景説明は後回しにする
2. **データドリブン**：主張には必ず数値・出典・根拠を添える。「おそらく」「たぶん」は使わない
3. **簡潔に**：1メッセージ・1スライド・1セクションに盛り込むポイントは最大3つまで
4. **行動に直結**：読んだ後に何をすべきかが明確になるよう、必ず「ネクストアクション」を記載する
5. **受け身表現を避ける**：「〜と思われます」より「〜です」「〜してください」を使う
6. **専門用語は最小限**：クライアントや外部向け資料では、業界用語を使う場合は括弧内に平易な言葉を補足する
7. **感情表現は控えめに**：過剰な敬語・クッション言葉・不要な謝罪表現を排除する

## 適用例
- CEO向けの日次サマリーは「今日の優先アクション3件」から始める
- 提案書の冒頭は「本提案により、○○を○週間・○○円で実現します」と結論を置く
- Slackや社内メモでは箇条書きを優先し、長文段落は使わない
