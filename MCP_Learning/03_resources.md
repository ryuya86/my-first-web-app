# MCPのリソース (Resources)

## リソースとは
MCPサーバーが公開するデータやコンテンツのことです。
ツールとは異なり、リソースは読み取り専用のデータです。

## リソースの種類
- **テキストリソース**: ドキュメント・設定ファイル
- **バイナリリソース**: 画像・PDF・その他ファイル
- **動的リソース**: リアルタイムで変化するデータ

## URI形式
リソースはURI（Uniform Resource Identifier）で識別されます。
例: `file:///path/to/document.txt`

## 活用シーン
- プロジェクトのドキュメントを読み込む
- 設定ファイルの内容を参照する
- データファイルを分析する
