# タスク: 夕方のサマリー

## 目的
今日1日の活動をまとめてSlackに報告する。

## 手順

### Step 1: 今日の統計を取得
`python tools/db.py stats` で統計を取得。

### Step 2: サマリーを作成
```
🌙 お疲れさまです、タクミ

📊 今日の活動
- 新着案件: X件発見 → Y件通知済み
- メッセージ: X件確認 → Y件返信案作成
- エラー: X件（あれば）

📈 今週の進捗
- 応募数: X件
- 返信率: XX%

明日もよろしくお願いします 💪
```

### Step 3: Slackに送信
```
python tools/slack_notify.py report '{"title":"🌙 夕方のサマリー","body":"内容..."}'
```
