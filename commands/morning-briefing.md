# タスク: 朝のブリーフィング

## 目的
CEOが朝起きたときに、昨日の成果と今日の予定を一目で把握できるレポートをSlackに送信する。

## 手順

### Step 1: 昨日の統計を取得
`python tools/db.py stats` で直近30日の変換率統計を取得する。
`python tools/db.py stats-category` でカテゴリ別統計を取得する。

### Step 2: ブリーフィングを作成
以下の構成でレポートを作成する:

```
☀️ おはようございます、タクミ

📊 直近の成績
- 応募: X件 → 返信: Y件 → 契約: Z件
- 売上: ¥XX,XXX
- 勝率: XX%

📋 今日のスケジュール
- 9:00 / 13:00 / 18:00: 案件収集
- 3時間おき: メッセージ巡回

💡 気づき
（統計から読み取れる改善点があれば記載）
```

### Step 3: Slackに送信
```
python tools/slack_notify.py briefing '{"title":"☀️ 朝のブリーフィング","body":"内容..."}'
```
