01_introduction.md

MCPとは？ (Model Context Protocol)

概要

MCP（Model Context Protocol）は、Anthropicが開発したオープンスタンダードです。AIモデルと外部ツール・データソースを安全に接続するためのプロトコルです。

主な特徴

  • AIアシスタントがリアルタイムでデータにアクセス可能
  • 標準化されたインターフェースで様々なツールを統合
  • セキュアな通信プロトコル

活用例

  • ファイルシステムへのアクセス
  • データベースの読み書き
  • 外部APIとの連携
02_tools.md

MCPのツール (Tools)

ツールとは

MCPサーバーが提供する機能の単位です。AIモデルはツールを呼び出すことで、外部の操作を実行できます。

主なツールの種類

  • ファイル操作: 読み込み・書き込み・検索
  • コード実行: スクリプトの実行・テスト
  • Web検索: インターネット上の情報収集
  • データベース: クエリの実行・データ管理

ツールの定義

各ツールはJSONスキーマで入力パラメータを定義します。AIモデルはスキーマを参照して正しい形式でツールを呼び出します。

03_resources.md

MCPのリソース (Resources)

リソースとは

MCPサーバーが公開するデータやコンテンツのことです。ツールとは異なり、リソースは読み取り専用のデータです。

リソースの種類

  • テキストリソース: ドキュメント・設定ファイル
  • バイナリリソース: 画像・PDF・その他ファイル
  • 動的リソース: リアルタイムで変化するデータ

URI形式

リソースはURI（Uniform Resource Identifier）で識別されます。例: file:///path/to/document.txt

活用シーン

  • プロジェクトのドキュメントを読み込む
  • 設定ファイルの内容を参照する
  • データファイルを分析する
04_setup.md

MCPのセットアップ

必要なもの

  • Node.js (v18以上)
  • Claude Desktop または Claude Code

インストール手順

  1. MCPサーバーパッケージをインストール
    npm install @modelcontextprotocol/server-filesystem
  2. 設定ファイルを作成 (claude_desktop_config.json)
  3. Claudeを再起動してMCPサーバーを有効化

動作確認

Claudeとの会話でMCPツールが使えることを確認しましょう。
